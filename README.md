# EventSphere – Event Management Portal

**EventSphere** is an extremely modern, visually attractive, animated, and responsive frontend static website designed for a Web Technologies lab assignment.

## 🚀 Features
- **Premium UI/UX:** Dark theme with neon blue/purple gradients, glassmorphism, and hover effects.
- **Animations:** Scroll reveal animations, animated counters, smooth scrolling, and dynamic typing/transitions.
- **Responsive Design:** Fully mobile-friendly layout built with Flexbox and CSS Grid.
- **5-Page Navigation:** Home, Events, Register, About, Contact.
- **Interactive Forms:** Client-side JavaScript validation for Registration and Contact forms.
- **Modern Assets:** Font Awesome icons, Google Fonts (Outfit), and beautiful placeholder images.

## 💻 Technologies Used
- **HTML5:** Semantic structure (`<header>`, `<nav>`, `<section>`, `<footer>`).
- **CSS3:** Advanced animations, Variables (`:root`), Flexbox, Grid, Media Queries, Glassmorphism.
- **JavaScript (Vanilla):** DOM Manipulation, Form Validation, Intersection Observer (scroll animations), Interactive UI components.

## 📋 Project Report

**Domain Selected:** Event Management Portal
**Project Title:** EventSphere

### Page Structure Overview
1. **Home Page (`index.html`):** Features an animated hero banner, featured event cards, and dynamic statistics counters.
2. **Events Page (`events.html`):** Displays a grid of events with a functional-looking search bar and category filters.
3. **Registration Page (`register.html`):** Contains a stylish glassmorphism form for users to register for specific events with JS validation.
4. **About Page (`about.html`):** Highlights the mission, vision, a CSS-styled animated timeline, and technologies used.
5. **Contact Page (`contact.html`):** Features contact details with hover effects, an embedded Google Map, a contact form, and an interactive FAQ accordion.

### Setup Instructions
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No server setup is required since this is a static frontend website.
